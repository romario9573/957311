# Baseline выравнивание логотипа

## ✅ Выполнено: 17 января 2026

### Задача
Выровнять приставку [AI] по нижней границе (baseline) с основным текстом "Romario"

### Проблема
- [AI] визуально был ниже текста "Romario"
- Нижние границы текстов не на одном уровне
- Выравнивание по центру работало неправильно для текстов

### Решение

#### 1. Контейнер .logo-text
```css
.logo-text {
  display: inline-flex;
  align-items: baseline; /* Все элементы на одной базовой линии */
  line-height: 1;
}
```

#### 2. Все текстовые элементы
```css
.logo-base,
.logo-i-letter,
.logo-ai {
  vertical-align: baseline; /* Выравнивание по базовой линии */
  line-height: 1;           /* Убирает лишние отступы */
  display: inline-block;
}
```

#### 3. Приставка [AI] с микрокоррекцией
```css
.logo-ai {
  vertical-align: baseline;
  line-height: 1;
  position: relative;
  bottom: 0px; /* Можно настроить при необходимости */
}
```

### Ключевые изменения
1. ✅ `.logo-text` использует `align-items: baseline`
2. ✅ Все элементы имеют `vertical-align: baseline`
3. ✅ Все элементы имеют `line-height: 1`
4. ✅ Добавлена возможность микрокоррекции через `position: relative; bottom: 0px`

### Результат
✅ Нижние границы "Romario" и "[AI]" на одной линии  
✅ Выравнивание по базовой линии шрифта  
✅ [AI] визуально на правильной высоте  
✅ Шестеренка остаётся на месте  
✅ Логотип выглядит профессионально и аккуратно  
✅ Работает на всех размерах экрана

### Визуальная схема
```
ДО (align-items: center):
     ┌─────────┐
     │ Romario │ [AI]  ← [AI] по центру, ниже базовой линии
     └─────────┘ ─┘
     ────────────── (базовая линия не совпадает)

ПОСЛЕ (align-items: baseline):
           Romario [AI]   ← Всё выровнено по нижней границе
     ────────────── (общая базовая линия)
```
