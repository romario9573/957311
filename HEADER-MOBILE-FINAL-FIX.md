```css
  display: none !important;
}

/* КРИТИЧЕСКИ ВАЖНО: Скрываем Telegram кнопку на мобильных */

  display: none !important;
}

/* КРИТИЧЕСКИ ВАЖНО: Показываем гамбургер на мобильных */

  display: flex !important;
}
```

```css

```

```
│                                             │
│              О себе                         │

│              Контакты                       │
│                                             │
│         📱 Написать в Telegram              │
│                                             │
```

**Характеристики меню:**
- Полноэкранное (от header до низа)
- Темный фон: `rgba(15, 15, 15, 0.98)`
- Blur эффект: `blur(24px)`
- Анимация: slideDown (0.35s)
- Центрированный контент (max-width: 480px)
- Кнопки: 20px шрифт, 52px высота
- Telegram кнопка: градиент, выделена
