## 🎯 Анимации

### 1. Свечение "БЕСПЛАТНО"
```css
@keyframes glow {
  0%, 100% {
    text-shadow: 0 0 12px rgba(218, 164, 40, 0.6);
  }
  50% {
    text-shadow: 0 0 20px rgba(218, 164, 40, 0.8),
                 0 0 30px rgba(218, 164, 40, 0.5),
                 0 0 40px rgba(218, 164, 40, 0.3);
  }
}
```
Цикл: 2 секунды

### 2. Появление секции
- Fade in + translateY
- Duration: 800ms
- Easing: ease-out

### 3. Появление элементов
- Промо-блок: delay 200ms
- Таблица/карточки: delay 400ms
- Карточки: staggered delay (100ms каждая)

### 4. Hover эффекты
- Промо-блок: translateY(-4px)
- Строки таблицы: scale(1.01)
- Transitions: 300ms
